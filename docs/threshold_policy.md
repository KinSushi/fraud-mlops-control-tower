# Threshold Policy

## Default

The default classification threshold is `0.50`.

## Why threshold matters

In anomaly and risk workflows, threshold selection controls the trade-off between false positives and false negatives.

## Public portfolio policy

This repository demonstrates threshold mechanics only. It does not define a production risk policy.
